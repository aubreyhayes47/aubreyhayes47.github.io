Evidence — Case Briefs

This course currently has no briefs. Add briefs by creating a Markdown file named briefs/evidence/Case_Name.md (use underscores in filenames, e.g., Smith_v_Jones.md), include Jekyll front matter (title, citation, court, year, permalink, tags), and commit as a branch + PR for review.

How to add a brief

  1. Create a Markdown file at briefs/evidence/<Case_File>.md with front matter. The first paragraph after front matter becomes the summary shown on this index.
  2. Run the index generator locally (optional): node ./scripts/generate_indexes.js to update this course index, or let the repository workflow generate it after merging.
  3. Commit the file to a branch and open a Pull Request so maintainers can review.

If you'd like an in-page search box once briefs are present, include {% raw %}{% include briefs_search.html %}{% endraw %} at the top of the generated briefs/evidence/index.md (the repository's JS at /assets/js/briefs_search.js provides client-side filtering).
